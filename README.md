# Mumin Ul Bari — Cybersecurity Portfolio

A personal portfolio website focused on cybersecurity, cryptography, and IT skills.

## 🚀 Deploy to GitHub Pages

1. Create a new GitHub repository named **`<your-username>.github.io`**
   - Example: `MuminUlBari.github.io`

2. Upload the `index.html` file to the root of the repository.

3. Go to **Settings → Pages** in your repository.

4. Under **Source**, select **Deploy from a branch** → `main` → `/ (root)`.

5. Click **Save**. Your site will be live at:
   ```
   https://<your-username>.github.io
   ```

## 📁 File Structure
```
your-repo/
└── index.html       ← The entire portfolio (single file)
```

## ✏️ Customization Tips

- **Add a profile photo**: Replace the hex SVG decoration area with an `<img>` tag.
- **Update links**: Search for `github.com/MuminUlBari` and replace with your actual GitHub.
- **Add more projects**: Copy a `.project-card` block and fill in your details.
- **Add a PDF resume**: Add a download button linking to a `resume.pdf` placed in the same repo folder.

## 🎨 Tech Stack
- Pure HTML5 + CSS3 + Vanilla JS
- Google Fonts (Orbitron, Rajdhani, Share Tech Mono)
- No frameworks, no dependencies — loads fast everywhere
